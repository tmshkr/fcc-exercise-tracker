# Exercise Tracker GraphQL API

#### A microservice project, based on Free Code Camp's curriculum

### User Stories

1. I can use GraphiQL at [/graphql](https://tmshkr-fcc-exercise-tracker.glitch.me/graphql) to interact with the database.
1. I can view a list of users at [/users](https://tmshkr-fcc-exercise-tracker.glitch.me/users).
1. I can add a user at the top of the users list, and when attempting to add a username that is already in use, the input displays an error message, indicating that the username is taken.
1. I can view a list of a user's exercises by selecting their name from the users list.
1. I can view a user's exercise list by navigating to [/user/:username/exercises](https://tmshkr-fcc-exercise-tracker.glitch.me/user/hello/exercises).
1. I can add a new exercise for a user at the top of their exercise list.
